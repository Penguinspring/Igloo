1) 하이퍼텍스트(hypertext)의 개념과 특징

 하이퍼텍스트란 말 그대로 정상적인 텍스트를 초월한 텍스트를 말한다. 이것은 개별 정보들을 링크를 이용해 유기적으로 연결하여 비연속적, 비선형적 체계로 구성된 전자적 텍스트라고도 한다.텍스트의 선형적, 연속적 구조와 달리 비연속적, 비선형적으로 구성된 하이퍼 텍스트는 사용자에게 여러 방향을 제공한다. 필요한 정보를 텍스트 안에서 선택해 얻을 수 있다. 또한, 이미지나 사운드와 같은 여러 구성 요소들을 포함하기 때문에 기존의 텍스트와 달리 역동적이고 다양한 모습을 볼 수 있다. 정보들을 링크를 이용해 연결하는데, 노드와 링크를 단위로 하는 정보의 구조화 방식이라고 하이퍼텍스트를 설명할 수도 있다. 마지막으로 설계자와 사용자 간 상호작용을 가능하게 한다. 하이퍼텍스트 속에 있는 이미지나 도표를 수정하는 과정에서 사용자의 피드백을 받아 가면서 더 좋은 하이퍼텍스트를 작성할 수 있다.



2) 웹의 개념과 특징

 웹(Web)은 월드 와이드 웹(World Wide Web)을 줄여서 말하는 것이고, 다양한 형태의 데이터와 정보에 접근할 수 있도록 해 주는 인터넷 서비스이다. 기술적으로는 HTTP(HyperText Transfer Protocol)을 사용하는 네트워크이며, HTML(HyperText Markup Language)로 구성된 문서들과 문서들을 서로 연결해 주는 링크로 구성된 웹페이지의 집합체이다. 하이퍼텍스트를 이용하기 때문에 멀티미디어의 성질을 가진다. 1991년 최초의 웹을 개발한 팀 버너스-리는 국제적인 정보 및 자료 교환을 위해 웹 개발 프로젝트를 진행했고, 이후 웹의 보급이 이루어지며 보편적인 인터넷 서비스로 자리 잡았다. 인터넷 서비스로서의 역할이 중요해지면서 1993년 모자이크(Mosaic)와 웹 브라우저(Web Browser)의 보급을 시작으로 여러 웹이 나오기 시작했고, 현재는 크롬(Chrome)이 가장 많은 사용자가 이용하는 웹이다. 하이퍼텍스트와 마찬가지로 비선형성을 가지며, 멀티미디어의 역할을 하기 때문에 텍스트 이외의 시청각적인 요소들을 결합하여 사용자에게 전달 가능한 특징을 가지고 있다.



3) 웹2.0의 개념과 특징

 일방적 서비스 형태의 초기 웹(1.0)과 비교하여 기술적인 진보보다는 웹 서비스의 사용성을 대폭 개선하여 누구나 웹에 쉽고 편리하게 참여 및 정보를 공유하고 협력할 수 있도록 진화된 웹 환경을 웹 2.0이라고 할 수 있다. 웹의 개발 초기 당시 웹 1.0은 신문과 방송같이 기업들이 웹 페이지에 정보를 제공하고 사용자들은 정보를 얻는 일방적인 서비스였는데, 웹 2.0으로 발전하며 블로그와 같은 웹 플랫폼에 사용자가 직접 정보를 공유하며 상호작용할 수 있는 환경을 만들어 갈 수 있게 되었다. 대표적인 웹 2.0에는 위키피디아를 들 수 있는데, 사용자가 정보를 얻어 가면서 잘못된 정보를 수정할 수 있는 상호작용을 하며 집단 지성을 실천할 수 있다. 또한, 웹 2.0의 특징으로는 “참여, 공유, 개방” 의 세가지 키워드가 있는데, 웹 2.0의 상호작용을 통해 집단 지성을 실현하는 모습을 잘 나타내는 단어들이다.
